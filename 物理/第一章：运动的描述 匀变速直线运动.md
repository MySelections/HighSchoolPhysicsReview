## 匀变速运动基本公式  
位移公式：$x = v_{0}t + \frac{1}{2}at^2$  
速度公式：$v_{t} = v_{0} + at$  
  
速度位移公式：$v_{t}^{2} - v_{0}^{2} = 2ax$  
  
  
  
相同时间的位移差：$\Delta x = aT^2 \ , \ x_{m} - x_{n} = (m-n)aT^2$  
  
中间时刻速度：$v_{\frac{t}{2}} = \frac{v_{0} + v_{t}}2{} = \bar{v}$  
  
位移中点速度：$v_{\frac{x}{2}} = \sqrt{\frac{v_{0}^{2} + v_{t}^{2}}{2}}$  
  
  
  
## 自由落体运动基本公式  
  
速度公式：$v = gt$  
  
位移公式：$h = \frac{1}{2}gt^2$  
  
速度位移公式：$v^{2} = 2gh$
